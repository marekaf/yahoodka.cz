Kostarika – Sanatorium Durán

Původně to bylo sanatorium se všemi vlastnostmi plnohodnotné nemocnice schopné pojmout kolem 300 nemocných. Bylo postaveno za účelem léčby pacientů trpící tuberkulózou (peste blanca).
